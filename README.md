# Safe-Authenticator
SWH Arduino Project

CREATED BY @MUJaara, @lethalsnake1337, @BorisKalika

This Arduino Project is a safe mechanism. 

<h1> REQUIREMENTS </h1>
1. ESP32-WROOM-32
2. Blynk account and it's library
3. 1 Button Module 
4. 1 Power Supply (any 3V PWS works)
5. 1 7805 5v Regulator
6. 12V Electronic lock
